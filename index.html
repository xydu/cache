add hessian serialize for xmemcache client. guide: you can change xmemcache client default serialize like this. ip:port 

        <bean class="com.cache4j.transcoder.HessianSerializingTranscoder"/>
    </property>
    <property name="failureMode" value="false" />
</bean>

if you don't want change xmemecache default serialize, you can used by single method. eg: memcachedClient.setWithNoReply(key, DEFAULT_EXPER_TIME, value, definedTranscoder); the parameter definedTranscoder is defined by you, and it must implements Transcoder;

also you can use cache4j util class (CacheClient.java)
it has get and set method.
